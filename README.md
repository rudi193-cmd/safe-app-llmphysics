# safe-app-llmphysics

**r/LLMPhysics Competition Paper Judge**

A SAFE-native judging tool for the r/LLMPhysics community competition.

## What it does

Scores submitted physics papers against the official competition rubric:
- Research Question (20 pts)
- Citations (15 pts)
- Minimal Assumptions (20 pts)
- Scientific Concept (25 pts)
- Execution (20 pts)

Total: 100 points

## Usage

Open `site/index.html` in a browser. Uses your API key (Gemini, Groq, OpenAI, or most others) for scoring. No server required.
